Cheat Sheet Maker

从网页/PDF/粘贴内容摄入知识库，通过 RAG 检索 + LLM 生成考试大纲与小抄，并自动渲染为 PDF。

查看演示

1) 摄入

扫描网页 / 粘贴文本 / 上传 PDF，写入向量库（按用户隔离）。

+ Scan & Add to KB
+ Upload PDF & Add to KB

2) 生成大纲

使用 syllabus / 课程信息作为 query，结合 RAG 上下文提取考试主题列表。

Generating Outline... Est: ~32s

3) 生成 PDF 小抄

选主题后生成结构化 JSON，Headless 渲染为 HTML，Playwright 打印为 PDF。

Two-column A4 • KaTeX • Print CSS

说明

  • 此页面为纯静态展示用，不依赖后端/扩展，可直接给面试官演示产品形态。
  • Headless Renderer 仍由 React 的 #/print 路由负责。
