# Cheat Sheet RAG Clipper

Chrome 扩展，用于将网页内容保存到 RAG 知识库。

## 安装步骤

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"（右上角开关）
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome-extension` 文件夹

## 使用方法

1. 访问任意网页
2. 点击浏览器工具栏中的扩展图标
3. 在弹出窗口中点击 "Save to Knowledge Base" 按钮
4. 等待保存完成

## 功能说明

- 自动抓取当前网页的标题和主要内容
- 将内容发送到后端 API (`http://127.0.0.1:8000/api/rag/ingest`)
- 显示保存状态和结果

## 注意事项

- 确保后端服务已启动并运行在 `http://127.0.0.1:8000`
- 扩展需要访问后端 API 的权限（已在 manifest.json 中配置）

## 图标文件

如果需要自定义图标，请准备以下尺寸的 PNG 图片：
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

如果没有图标文件，扩展仍然可以正常工作，只是会显示默认图标。
